**Discussed Infra for VRIoT**  

Deployment Servers: Ubuntu server 16.04 LTS  
Docker/Kubernetes/LXC/VMWare/Vagrind

DB Layer: MongoDB 3.2  
Cache Engine: Redis/MemSQL  
Web UI: Angular/React  
Infra Monitoring: New Relic
